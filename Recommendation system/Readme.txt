Firstly, you open the command window here by using "shift + right click on mouse and select open command window here".
Now, give the command "javac *.java"
After the control gets to a new line, give the command "java Apriori"
Then, it will ask for "minimum support" count and the "k" value. give the above in order by pressing enter.
Now, the program will be executed and the progam execution time will be written on the console in seconds and the output will be generated in file which will be created, if already not present, with the name as "out_s=minimumSupport_k=Kvalue+". For Ex: for minimum support count=3 and k=8, the name of output file will be "out_s=3_k=8+".
Now open the file and check the ouput.
